# India Reaction Formats

These are reusable reaction-video formats for large-scale UGC campaigns.

They are designed around:
- raw phone footage
- explicit camera logic
- strong first-2-second hooks
- scalable variation across settings and characters
- low dependency on exact recurring-character continuity

Each format defines:
- what the camera is doing
- what the emotional arc is
- what kind of expression control we want
- why it scales
- how to prompt it consistently

Current formats:
- `format_01_picked_up_phone_reaction.json`
- `format_02_propped_desk_confession_reaction.json`
- `format_03_bed_glow_relief_reaction.json`
- `format_04_friend_show_screen_reaction.json`
