# India Raw Footage Experiments

This subfolder is the visible workspace for India-market raw-footage testing.

Keep here:
- scene experiments
- reference-image packs
- Sora vs Veo comparisons
- dialogue-delivery tests
- consistency failures and retries
- notes on what should become a real skill later

## Why this exists

The current work is still in practice mode.

We are learning:
- which prompts create believable Indian student footage
- which models hold identity better
- where moderation blocks happen
- how to structure clips for install-focused UGC
