# Start Here

This repo is not just code.

It is the working studio for Professor Curious ad creation:
- research
- hooks
- prompt experiments
- reaction-video systems
- India and US ad formats

If you are non-technical, start with these files.

## Best front door
- `CREATIVE_DASHBOARD.md`

## If you want to quickly understand the repo
- `README.md`

## If you want to see the latest experiments
- `experiments/LATEST.md`

## If you want to understand the product and the user
- `research/README.md`
- `research/product/PROF_CURIOUS_CREATIVE_RESEARCH_V2_2026-03-12.md`

## If you want to understand the India raw-footage system
- `india-raw-footage/README.md`

## If you want to know what changed recently
- `CONTEXT.md`

## The simplest mental model

Think of the repo like this:

### `research/`
Why the product matters.

### `experiments/`
What we are testing right now.

### `india-raw-footage/`
How we are building realistic India-native raw-footage ads.

### `ugc-street-interview/`
India street-style social-proof ads.

### `ugc-us-college-interview/`
US campus-style social-proof ads.

### `ugc-reaction/` and `experiments/reaction-pipeline/`
Reaction-video ad systems.

## Important note

The repo mainly stores:
- prompts
- scripts
- research
- structures
- experiment setups

Most generated videos stay local and are usually not committed into Git, so the repo stays clean and reusable.
